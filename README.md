
# AutoApplyMAX

## Description

AutoApplyMAX is a browser extension designed to automate the process of applying for jobs or other applications online. It simplifies the application process by automatically filling in forms and submitting them, saving you time and effort.

## Features

- Automatically fills out online application forms.
- Submits applications with a single click.
- Customizable to work with various application websites.

## Installation

1. Clone this repository to your local machine.
2. Open your web browser and navigate to the extensions page (e.g., `chrome://extensions` for Google Chrome).
3. Enable "Developer mode".
4. Click on "Load unpacked" and select the `src` directory from this repository.

## Usage

Once installed, the AutoApplyMAX icon will appear in your browser's toolbar. Navigate to a supported application website, and the extension will automatically detect the application form. Click the "Auto Apply" button to fill out and submit the form.
